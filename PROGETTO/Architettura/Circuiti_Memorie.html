CIRCUITI LOGICI

  • Sequenziali: l'output dipende SOLO dagli input
  • Combinatori: l'output dipende anche dallo stato precedente

Lo scopo è produrre un'output in relazione agli input dati

I circuiti si ottengono tramite porte logiche interconnesse tra loro, dando origine a delle reti logiche

Porte Logiche

CIRCUITI DI BASE

  • MULTIPLEXER: N segnali di controllo (S), 2^N input, 1 output.

  • DEMULTIPLEXER: N segnali di controllo (S), 1 input, 2^N output
  • DECODER: N input, 2^N output
  • ENCODER: 2^N input, N output

CIRCUITI MATEMATICI

  • COMPARATORE a N bit
  • SOMMATORI
    MEZZO SOMMATORE SOMMATORE COMPLETO
  • SHIFTER

RITARDI DEI CIRCUITI

I circuiti logici possiedono un ritardo nella produzione del risultato, attorno a 0.1ns

Questo ritardo è influente, basti pensare che il periodo di CLK si aggira attorno a 1ns se F=1Ghz
